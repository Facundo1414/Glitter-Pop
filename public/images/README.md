## Tips para las imágenes:

1. **Formato**: Usa JPG para fotos, PNG para logos
2. **Tamaño**: Optimiza antes de subir (max 500KB por imagen)
3. **Nombres**: Usa nombres descriptivos en minúsculas con guiones
4. **Calidad**: Mantén buena resolución pero comprime para web

### Herramientas de optimización:

- https://tinypng.com/
